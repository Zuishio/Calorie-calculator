# Calorie-calculator
Простое десктопное приложение для учета калорий и расчета пищевой ценности порций. Проект разрабатывается в рамках учебного задания.


## Установка и запуск
1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение: `python main.py`
