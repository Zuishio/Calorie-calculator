# Calorie-calculator
Простое десктопное приложение для учета калорий и расчета пищевой ценности порций. Проект разрабатывается в рамках учебного задания.


## Возможности
- Автоматический пересчет калорий, белков, жиров и углеводов в режиме реального времени при изменении веса продукта.
- Автоматический подсчет итоговой суммы калорий для всех продуктов в таблице, а также расчет калорийности на одну порцию (с учетом количества людей)
- Поддержка темной и светлой темы 
- Поддержка горячих клавиш (Ctr+T - смена цветовой темы, ctrl+delete - очистка таблицы)
- сохранение своего меню и импорт уже готового ( в процессе разработки)
- Поддержка импорта фотографий блюд или продуктов (.png .jpg .jpeg .bmp)


## Требования к программе
Для корректной работы декстопного приложения на вашем компьютере должны быть установлены:
- Операционная система: Windows 10/11,MacOS или Linux
- Python версии 3.10 или же выше, а также её библиотеки:
- Pyqt6
- sqlite3
- json
- os
- sys
  

## Установка и запуск
1.Клонируйте репозиторий к себе на компьютер или скачайте архив с кодом и распакуйте его.
2. Создайте и активируйте виртуальное окружение.
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите приложение: `python main.py`
